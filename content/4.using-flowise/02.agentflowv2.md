* **–ù–∞–∑–≤–∞–Ω–∏–µ —Ñ–æ—Ä–º—ã, –û–ø–∏—Å–∞–Ω–∏–µ —Ñ–æ—Ä–º—ã, –¢–∏–ø—ã –ø–æ–ª–µ–π —Ñ–æ—Ä–º—ã (Form Title / Description / Input Type):** –ü—Ä–∏ –≤—ã–±–æ—Ä–µ –≤–≤–æ–¥–∞ —á–µ—Ä–µ–∑ —Ñ–æ—Ä–º—É –ø–æ–∑–≤–æ–ª—è—é—Ç –Ω–∞—Å—Ç—Ä–æ–∏—Ç—å –≤–Ω–µ—à–Ω–∏–π –≤–∏–¥ —Ñ–æ—Ä–º—ã –¥–ª—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è, –∑–∞–¥–∞–≤ —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Ç–∏–ø—ã –ø–æ–ª–µ–π —Å –ø–æ–¥–ø–∏—Å—å—é –∏ –∏–º–µ–Ω–∞–º–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö.
* **–≠—Ñ–µ–º–µ—Ä–Ω–∞—è –ø–∞–º—è—Ç—å (Ephemeral Memory):** –ü—Ä–∏ –≤–∫–ª—é—á–µ–Ω–∏–∏ —É–∫–∞–∑—ã–≤–∞–µ—Ç —Ä–∞–±–æ—á–µ–º—É –ø—Ä–æ—Ü–µ—Å—Å—É –Ω–∞—á–∞—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ, –∏–≥–Ω–æ—Ä–∏—Ä—É—è –≤—Å–µ –ø—Ä–µ–¥—ã–¥—É—â–∏–µ —Å–æ–æ–±—â–µ–Ω–∏—è –∏–∑ –¥–∏–∞–ª–æ–≥–∞, —Ç–æ –µ—Å—Ç—å —Å —á–∏—Å—Ç–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –ø–∞–º—è—Ç–∏.
* **–°–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–∞–±–æ—á–µ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞ (Flow State):** –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç –ø–æ–ª–Ω—ã–π –Ω–∞–±–æ—Ä –Ω–∞—á–∞–ª—å–Ω—ã—Ö –ø–∞—Ä –∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è —Å–æ—Å—Ç–æ—è–Ω–∏—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è $flow.state. –í—Å–µ –∫–ª—é—á–∏ —Å–æ—Å—Ç–æ—è–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –±—É–¥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∏–ª–∏ –æ–±–Ω–æ–≤–ª—è—Ç—å—Å—è –ø–æ—Å–ª–µ–¥—É—é—â–∏–º–∏ —É–∑–ª–∞–º–∏, –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –æ–±—ä—è–≤–ª–µ–Ω—ã –∏ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω—ã –∑–¥–µ—Å—å.

**–í—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ:** –ü—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞—á–∞–ª—å–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ, –∑–∞–ø—É—Å–∫–∞—é—â–∏–µ —Ä–∞–±–æ—á–∏–π –ø—Ä–æ—Ü–µ—Å—Å ‚Äî —ç—Ç–æ –ª–∏–±–æ —Å–æ–æ–±—â–µ–Ω–∏–µ –∏–∑ —á–∞—Ç–∞, –ª–∏–±–æ –¥–∞–Ω–Ω—ã–µ, –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–µ —á–µ—Ä–µ–∑ —Ñ–æ—Ä–º—É.

**–í—ã—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ:** –ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π –≤—ã—Ö–æ–¥ –¥–ª—è –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –∫ –ø–µ—Ä–≤–æ–º—É —Ä–∞–±–æ—á–µ–º—É —É–∑–ª—É, –ø–µ—Ä–µ–¥–∞–≤–∞—è –Ω–∞—á–∞–ª—å–Ω—ã–µ –≤—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –∏ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–∞–±–æ—á–µ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞.

### **–£–∑–µ–ª LLM (LLM Node)**

–ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –ø—Ä—è–º–æ–π –¥–æ—Å—Ç—É–ø –∫ –Ω–∞—Å—Ç—Ä–æ–µ–Ω–Ω–æ–π –±–æ–ª—å—à–æ–π —è–∑—ã–∫–æ–≤–æ–π –º–æ–¥–µ–ª–∏ (LLM) –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –∑–∞–¥–∞—á –ò–ò, –ø–æ–∑–≤–æ–ª—è—è —Ä–∞–±–æ—á–µ–º—É –ø—Ä–æ—Ü–µ—Å—Å—É –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –≤—ã–ø–æ–ª–Ω—è—Ç—å –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö.


**–§—É–Ω–∫—Ü–∏–æ–Ω–∞–ª**: –≠—Ç–æ—Ç —É–∑–µ–ª –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –∑–∞–ø—Ä–æ—Å—ã –≤ LLM –Ω–∞ –æ—Å–Ω–æ–≤–µ –∑–∞–¥–∞–Ω–Ω—ã—Ö –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π (–°–æ–æ–±—â–µ–Ω–∏–π) –∏ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞. –û–Ω –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –¥–ª—è –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Ç–µ–∫—Å—Ç–∞, —Å—É–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, –ø–µ—Ä–µ–≤–æ–¥–∞, –∞–Ω–∞–ª–∏–∑–∞, –æ—Ç–≤–µ—Ç–æ–≤ –Ω–∞ –≤–æ–ø—Ä–æ—Å—ã –∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ JSON-–≤—ã–≤–æ–¥–∞ —Å–æ–≥–ª–∞—Å–Ω–æ –∑–∞–¥–∞–Ω–Ω–æ–π —Å—Ö–µ–º–µ. –ò–º–µ–µ—Ç –¥–æ—Å—Ç—É–ø –∫ –ø–∞–º—è—Ç–∏ —Ç–µ–∫—É—â–µ–≥–æ –¥–∏–∞–ª–æ–≥–∞ –∏ –º–æ–∂–µ—Ç —á–∏—Ç–∞—Ç—å/–∑–∞–ø–∏—Å—ã–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–∞–±–æ—á–µ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞ (Flow State).

–ü–∞—Ä–∞–º–µ—Ç—Ä—ã –Ω–∞—Å—Ç—Ä–æ–π–∫–∏:  

* **–ú–æ–¥–µ–ª—å:** –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç AI-–º–æ–¥–µ–ª—å –≤—ã–±—Ä–∞–Ω–Ω–æ–≥–æ —Å–µ—Ä–≤–∏—Å–∞ ‚Äî –Ω–∞–ø—Ä–∏–º–µ—Ä, GPT-4o –æ—Ç OpenAI –∏–ª–∏ Google Gemini.
* **–°–æ–æ–±—â–µ–Ω–∏—è (Messages):** –ó–∞–¥–∞—é—Ç –≤—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –¥–ª—è LLM –≤ –≤–∏–¥–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —Ä–æ–ª–µ–π ‚Äî –°–∏—Å—Ç–µ–º–∞, –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å, –ê—Å—Å–∏—Å—Ç–µ–Ω—Ç, –†–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫ ‚Äî –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –æ—Ç–≤–µ—Ç–æ–º –ò–ò. –í —Ç–µ–∫—Å—Ç –º–æ–∂–Ω–æ –≤—Å—Ç–∞–≤–ª—è—Ç—å –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–µ –¥–∞–Ω–Ω—ã–µ —á–µ—Ä–µ–∑ {{ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è }}.
* –ü–∞–º—è—Ç—å (Memory): –ü—Ä–∏ –≤–∫–ª—é—á–µ–Ω–∏–∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –¥–æ–ª–∂–µ–Ω –ª–∏ LLM —É—á–∏—Ç—ã–≤–∞—Ç—å –∏—Å—Ç–æ—Ä–∏—é —Ç–µ–∫—É—â–µ–≥–æ –¥–∏–∞–ª–æ–≥–∞ –ø—Ä–∏ —Ñ–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–∏–∏ –æ—Ç–≤–µ—Ç–∞.
- –¢–∏–ø –ø–∞–º—è—Ç–∏, –†–∞–∑–º–µ—Ä –æ–∫–Ω–∞, –ú–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ç–æ–∫–µ–Ω–æ–≤ (Type / Window Size / Max Token Limit): –ü—Ä–∏ –≤–∫–ª—é—á–µ–Ω–∏–∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –¥–æ–ª–∂–µ–Ω –ª–∏ LLM —É—á–∏—Ç—ã–≤–∞—Ç—å –∏—Å—Ç–æ—Ä–∏—é —Ç–µ–∫—É—â–µ–≥–æ –¥–∏–∞–ª–æ–≥–∞ –ø—Ä–∏ —Ñ–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–∏–∏ –æ—Ç–≤–µ—Ç–∞.
- –í—Ö–æ–¥—è—â–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ (Input Message): –£–∫–∞–∑—ã–≤–∞–µ—Ç –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –∏–ª–∏ —Ç–µ–∫—Å—Ç, –∫–æ—Ç–æ—Ä—ã–π –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è –∫–∞–∫ –ø–æ—Å–ª–µ–¥–Ω–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –≤ –∫–æ–Ω–µ—Ü —Ç–µ–∫—É—â–µ–≥–æ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞ –¥–∏–∞–ª–æ–≥–∞ (–≤–∫–ª—é—á–∞—è –Ω–∞—á–∞–ª—å–Ω—ã–π –∫–æ–Ω—Ç–µ–∫—Å—Ç –∏ –ø–∞–º—è—Ç—å) –ø–µ—Ä–µ–¥ –æ–±—Ä–∞–±–æ—Ç–∫–æ–π LLM/–ê–≥–µ–Ω—Ç–æ–º.
* –§–æ—Ä–º–∞—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–≥–æ –æ—Ç–≤–µ—Ç–∞ (Return Response As): –ù–∞—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç, –∫–∞–∫ –∫–ª–∞—Å—Å–∏—Ñ–∏—Ü–∏—Ä—É–µ—Ç—Å—è –≤—ã–≤–æ–¥ LLM ‚Äî –∫–∞–∫ —Å–æ–æ–±—â–µ–Ω–∏–µ –æ—Ç –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –∏–ª–∏ –ê—Å—Å–∏—Å—Ç–µ–Ω—Ç–∞ ‚Äî —á—Ç–æ –≤–ª–∏—è–µ—Ç –Ω–∞ –¥–∞–ª—å–Ω–µ–π—à—É—é –æ–±—Ä–∞–±–æ—Ç–∫—É –ø–∞–º—è—Ç–∏ –∏ –∂—É—Ä–Ω–∞–ª–∏—Ä–æ–≤–∞–Ω–∏–µ
* –°—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–π JSON-–≤—ã–≤–æ–¥ (JSON Structured Output): –î–∞—ë—Ç –∫–æ–º–∞–Ω–¥—É LLM —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞—Ç—å –æ—Ç–≤–µ—Ç —Å–æ–≥–ª–∞—Å–Ω–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–π JSON-—Å—Ö–µ–º–µ —Å –∫–ª—é—á–∞–º–∏, —Ç–∏–ø–∞–º–∏ –¥–∞–Ω–Ω—ã—Ö –∏ –æ–ø–∏—Å–∞–Ω–∏—è–º–∏, —á—Ç–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º–æ—Å—Ç—å –∏ —É–¥–æ–±–æ—á–∏—Ç–∞–µ–º–æ—Å—Ç—å –¥–ª—è –º–∞—à–∏–Ω.
* –û–±–Ω–æ–≤–ª–µ–Ω–∏–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è —Ä–∞–±–æ—á–µ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞ (Update Flow State): –ü–æ–∑–≤–æ–ª—è–µ—Ç —É–∑–ª—É –∏–∑–º–µ–Ω—è—Ç—å —Å–æ—Å—Ç–æ—è–Ω–∏–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è $flow.state –≤–æ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã, –æ–±–Ω–æ–≤–ª—è—è –∑–∞—Ä–∞–Ω–µ–µ –∑–∞–¥–∞–Ω–Ω—ã–µ –∫–ª—é—á–∏. –ù–∞–ø—Ä–∏–º–µ—Ä, –º–æ–∂–Ω–æ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Ä–∞–±–æ—Ç—ã —ç—Ç–æ–≥–æ —É–∑–ª–∞ –ø–æ–¥ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω—ã–º –∫–ª—é—á–æ–º –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –ø–æ—Å–ª–µ–¥—É—é—â–∏—Ö —É–∑–ª–æ–≤.


–í—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ: –£–∑–µ–ª –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –¥–∞–Ω–Ω—ã–µ –∏–∑ –Ω–∞—á–∞–ª—å–Ω–æ–≥–æ —Ç—Ä–∏–≥–≥–µ—Ä–∞ —Ä–∞–±–æ—á–µ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞ –∏–ª–∏ –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö –ø—Ä–µ–¥—ã–¥—É—â–∏—Ö —É–∑–ª–æ–≤, –≤–∫–ª—é—á–∞—è –∏—Ö –≤ –ø–æ–ª—è –°–æ–æ–±—â–µ–Ω–∏–π –∏–ª–∏ –í—Ö–æ–¥—è—â–µ–≥–æ —Å–æ–æ–±—â–µ–Ω–∏—è. –¢–∞–∫–∂–µ –º–æ–∂–µ—Ç –ø–æ–ª—É—á–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ $flow.state, –µ—Å–ª–∏ –≤—Ö–æ–¥–Ω—ã–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –Ω–∞ –Ω–µ–≥–æ —Å—Å—ã–ª–∞—é—Ç—Å—è.  

–í—ã—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ: –ì–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –æ—Ç–≤–µ—Ç LLM –≤ –≤–∏–¥–µ –æ–±—ã—á–Ω–æ–≥–æ —Ç–µ–∫—Å—Ç–∞ –∏–ª–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ JSON-–æ–±—ä–µ–∫—Ç–∞. –ö–∞—Ç–µ–≥–æ—Ä–∏—è –≤—ã–≤–æ–¥–∞ ‚Äî –ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å –∏–ª–∏ –ê—Å—Å–∏—Å—Ç–µ–Ω—Ç ‚Äî –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –Ω–∞—Å—Ç—Ä–æ–π–∫–æ–π –§–æ—Ä–º–∞—Ç –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–≥–æ –æ—Ç–≤–µ—Ç–∞.

### **–£–∑–µ–ª –∞–≥–µ–Ω—Ç–∞ (Agent Node)**

### **7. –£–∑–µ–ª Condition (Condition Node)**

### **14. –í—ã–ø–æ–ª–Ω–µ–Ω–∏–µ —É–∑–ª–∞ –ø–æ—Ç–æ–∫–∞ (Execute Flow Node)**
